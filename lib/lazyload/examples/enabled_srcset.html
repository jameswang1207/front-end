lazyload Plugin for jQuery | Zepto

This project is forked from tuupola/jquery_lazyload and add features below:
* Many Details for improve performance;
* IE6/7 support;
* Available with Zepto;
* `vertical_only` \ `minimum_interval` \ `use_minimum_interval_in_ios` \ `no_fake_img_loader` options.

More information on jieyou/lazyload.

Plugin enabled width `srcset`

About `srcset` : video|w3c .
Use `data-original-srcset` attribute instead of `srcset` , so images can lazyload .
If you want browsers not support `srcset` to work well , add `data-original` at the same time . 

$("img.lazy").lazyload();
<img class="lazy" data-original="img/bmw_m1_hood.jpg" data-original-srcset="img/bmw_m1_hood.jpg 765w" sizes="765px" width="765" height="574">
